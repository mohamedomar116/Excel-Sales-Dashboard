# 📊 Excel Sales Performance Dashboard 2024

## 📌 Overview
Interactive Excel dashboard analyzing 500 sales transactions 
across 6 regions in Egypt throughout 2024.

## ✨ Features
- 💰 KPI Cards: Total Sales, Profit, Orders & Profit Margin
- 📈 Monthly Sales & Profit Trend Chart
- 🛍️ Sales Breakdown by Product & Region
- 🔍 Interactive Slicers (Quarter, Category, Region)

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers & Conditional Formatting
